Estructura de Rmarkdown: YAML

Encabezado YAML (Yet Another Markup Language)

Este encabezado determina los parmetros generales de formato para el reporte a compilar. En lenguaje computacional se entiende como los metadatos del documento; esto es, informacin que define el formato del archivo resultante, ms no su contenido.

Este encabezado se crea automticamente cuando se selcciona el tipo de documento con el que se desea trabajar y siempre va entre - - -.

El output va a depender del tipo de documento que se selccione, el cual puede ser un html_document, word_document, pdf_document, ioslides_presentation, entre otros.

Estructura de Rmarkdown: YAML

  • Ttulo (title): Texto entre comillas que servir de ttulo general al documento.
  • Subttulo (subtitle): Texto entre comillas que servir de subttulo para el ttulo general del documento.
  • Autor (author): Texto entre comillas para indicar el nombre del o los autores.
  • Fecha (date): Campo para indicar la fecha. En el texto, con la expresin '\today', se solicita que imprima la fecha actual segn el calendario del sistema operativo.

Estructura de Rmarkdown: YAML

ioslides_presentation

---
title: "Herramientas Profesionales para Data Science - R Markdown"
author: "Carolina Molinare"
date: "7/12/2021"
output: 
  ioslides_presentation:
    widescreen: true
    smaller: true
    transition: default
    incremental: true
---

ioslides_presentation

Opciones Descripcin
widescreen: true Permite mostrar la presentacin utilizando un factor de forma ms amplio, usando la opcin de pantalla panormica

transition: default Permite modificar la velocidad de transicin de las diapositivas. Pueden ser default, slower, faster
incremental: true Permite que las listas vayan apareciento de forma incremental

Estructura de Rmarkdown: Chunks

Chunks o bloques de cdigos

Se puede insertar un chunk de las siguientes formas:

  1. Apretando Ctrl + Alt + I en Windows y Cmd + Alt + I en Mac

  2. Seleccionando aadir un chunk de la barra de edicin

Estructura de Rmarkdown: Chunks Options

Chunk Option Descripcin
echo=FALSE Previene que se imprima en el output la lnea de cdigo asociada a la funcin que ejecuta
eval=FALSE Evita que el cdigo sea evaluado (o se ejecute)
include=FALSE Se ejecuta el cdigo, pero sin mostrar los resultados ni el cdigo en el output
message=FALSE Previene que se impriman en el output mensajes asociados a la lnea de cdigo que ejecuta
warning=FALSE Previene que se impriman en el output warnings asociados a la lnea de cdigo que ejecuta
error=FALSE Se renderiza (o compila) aunque el cdigo devuelva un error

En el caso de RMarkdown los signos “gato” (#) indican que se est sealando un ttulo o subttulo de seccin. Mientras ms signos gatos se pongan se estar indicando que se trata de un ttulo de menor jerarqua.

Estructura de Rmarkdown: Texto

Tablas

La construccin de tablas en RMarkdown se realiza separando los elementos de cada columna con un tabulador vertical (|). Cada fila de texto corresponder a una fila de la tabla.

Se puede definir la alineacin del texto segn la distribucin de guiones debajo de los ttulos. Si los guiones exceden hacia ambos lados del texto, este quedar centrado. Si excede hacia uno u otro lado quedar alineado hacia la izquierda o la derecha.

Las reglas para destacar texto, incluir enlaces y notas al pie, tambin aplican las tablas. No es necesario separar cada fila con guiones, basta comenzar una nueva fila de texto.
